# 法律文书起草 (Draft)

该指令旨在辅助法律专业人士起草各类标准化法律文书。系统基于广泛的法律文书库和现行法律法规，生成格式规范、逻辑严密、用语专业的初稿。

> [!NOTE]
> 本指令辅助生成的文档仅作为**初稿**使用，最终签署版本必须经由执业律师审核。

## 核心功能 (Core Capabilities)

- **标准化模板调用**：基于预置的 Best Practice 模板生成文书。
- **智能条款适配**：根据交易背景自动推荐特定的合同条款（如 GDPR 合规条款）。
- **法律依据自动引用**：自动关联文书内容所涉及的法律法规条文。

## 使用方法 (Usage)

```bash
/draft [DocType] [KeyInfo...]
```

### 参数说明 (Input Parameters)

| 参数             | 必选 | 说明                               | 示例                               |
| :--------------- | :--- | :--------------------------------- | :--------------------------------- |
| `DocType`        | 是   | 文书类型代码或中文名称             | `律师函`, `NDA`, `补充协议`        |
| `KeyInfo`        | 是   | 关键背景事实、争议金额、各方名称等 | `关于XX公司拖欠货款50万`           |
| `--style`        | 否   | 文书语气风格                       | `strict` (严厉), `friendly` (友好) |
| `--jurisdiction` | 否   | 适用法律/管辖地                    | `CN` (中国大陆), `HK` (香港)       |

## 支持类型 (Supported Types)

- **律师函 (Legal Letter)**：包含催收函 (Collection Letter)、侵权告知函 (Cease & Desist)。
- **法律意见书 (Legal Opinion)**：针对具体法律问题的专业分析意见。
- **保密协议 (NDA)**：包含单向 (Unilateral) 及双向 (Mutual) 保密协议。
- **合同条款 (Contract Clauses)**：不可抗力、争议解决、知识产权归属等特定条款。
- **备忘录 (Memo)**：内部法律研究备忘录。

## 标准作业程序 (SOP)

系统执行本指令时，遵循以下标准流程：

1.  **需求解析 (Requirement Analysis)**
    - 提取关键事实 (Fact Extraction)：主体、客体、行为、因果关系。
    - 确定管辖 (Jurisdiction Check)：默认适用中华人民共和国法律。

2.  **框架构建 (Skeleton Construction)**
    - 调用对应的 `Template`。
    - 生成文书大纲结构（首部、正文、尾部）。

3.  **内容生成 (Drafting)**
    - 运用“法言法语”进行文本填充。
    - **引用核对**：对于引用的法律条文，通过 `/search` 指令验证其有效性。

4.  **自我审查 (Self-Review)**
    - 检查是否包含核心要素（如：律师函必须包含授权声明）。
    - 格式一致性检查。

## 质量控制 (Quality Control)

- **准确性**：引用的法律条文必须为现行有效版本。
- **完整性**：必须包含文书类型所需的全部必备条款（Essential Clauses）。
- **专业性**：避免使用口语化表达，保持法律文书的严肃性。
