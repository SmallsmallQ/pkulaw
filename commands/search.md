# 法律法规与案例检索 (Search)

该指令用于调用权威法律数据库（如北大法宝），执行精准的法律信息检索。系统支持自然语言查询，并能自动筛选现行有效的法规和具有指导意义的案例。

## 核心功能 (Core Capabilities)

- **跨库检索**：覆盖中央/地方法规、司法解释、指导性案例、裁判文书、法学期刊等。
- **效力过滤**：自动识别法规的时效性（现行有效/失效/已被修改）。
- **智能排序**：基于关联度、效力级别（法律 > 行政法规 > 部门规章）和发布日期进行排序。

## 使用方法 (Usage)

```bash
/search [Query] [--scope <database>] [--filter <condition>]
```

### 参数说明 (Input Parameters)

| 参数               | 说明                     | 示例                                             |
| :----------------- | :----------------------- | :----------------------------------------------- |
| `Query`            | 检索关键词或自然语言问题 | `未成年人网络保护`, `第110号指导案例`            |
| `--scope`          | 限定检索范围             | `laws` (法规), `cases` (案例), `journals` (期刊) |
| `--effective_only` | 仅显示现行有效的法规     | `true` (默认)                                    |

## 示例 (Examples)

- **法规检索**：`/search 关于借贷利率上限的最新规定 --scope laws`
- **案例检索**：`/search 竞业限制 补偿金 标准 --scope cases`
- **精准定位**：`/search 最高人民法院关于适用民法典合同编的解释`

## 标准作业程序 (SOP)

1.  **意图识别 (Intent Recognition)**
    - 分析用户查询是寻找具体的法条、案例还是法律概念。
    - 提取核心关键词 (Keywords) 和限定条件 (Constraints)。

2.  **构建检索策略 (Search Strategy)**
    - 确定目标数据库（法规库 vs 案例库）。
    - 构建布尔逻辑查询式（如：`(竞业限制 OR 竞业禁止) AND 补偿`）。

3.  **结果处理 (Result Processing)**
    - **效力核查**：标记每一条法规的效力状态（有效、失效、部分失效）。
    - **摘要生成**：提取与查询最相关的段落。

4.  **引用输出 (Citation)**
    - 按照标准法律引用格式输出结果（如：《中华人民共和国民法典》第一千二百三十四条）。
    - 提供原文链接（如有）。

## 质量控制 (Quality Control)

- **时效性**：优先返回最新的法律法规。
- **权威性**：案例检索优先展示“指导性案例”和“公报案例”，其次为高院/中院的生效判决。
- **准确性**：杜绝“幻觉”引用，所有法条必须真实存在。

## 免责声明 (Disclaimer)

检索结果依赖于外部数据库的完整性和更新频率。对于尚未收录的最新规定或特殊地方法规，系统可能无法提供。请以政府公报或法院正式发布的文本为准。
