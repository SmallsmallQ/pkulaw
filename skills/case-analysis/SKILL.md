# 案例分析技能 (Case Analysis)

本技能用于对法院判决书进行结构化分析，帮助用户快速抓住案件核心。

## 分析方法论：FIRAC (适配版)

### 1. 核心事实 (Facts)

- **争议焦点事实**：剔除无关细节，仅保留与判决结果有因果关系的事实。
- **程序经过**：一审、二审、再审的简要过程。

### 2. 争议焦点 (Issues)

- 法院归纳的争议焦点是什么？
- 双方当事人的核心抗辩观点是什么？

### 3. 裁判规则 (Rule/Reasoning)

- **法律依据**：法院引用了哪些具体的法律条文？
- **说理逻辑**：法院如何将法律适用于案件事实？法院的解释逻辑是什么？
- **关键突破**：本案是否创设了新的规则，或对既有规则进行了扩大/限缩解释？

### 4. 判决结果 (Conclusion/Holding)

- 最终的胜败诉情况及其具体内容。
